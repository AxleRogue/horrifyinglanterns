# Issue Tracker & Guidelines

Welcome to the **Horrifying Lanterns** issue tracker! We appreciate your help in making this mod better. To ensure a productive environment for everyone, please follow the guidelines below.

## Useful Links
- **Official Issue Tracker:** [GitHub Issues](https://github.com/AxleRogue/horrifyinglanterns/issues)
- **New to Open Source?** Check out the [Open Source Guide](https://opensource.guide) for tips on how to contribute effectively.

---

## Issue Labels
When browsing or opening issues, you will see the following labels. These help us categorize and prioritize work:

| Label | Description |
| :--- | :--- |
| `ability` | Related to item or entity abilities. |
| `api` | Issues or requests concerning the Mod API. |
| `bug` | Something is broken or not working as intended. |
| `cooldowns` | Specifically for issues regarding ability or item cooldowns. |
| `documentation` | Improvements, corrections, or additions to the wiki and docs. |
| `enhancement` | Suggestions for new features or improvements to existing ones. |
| `entities` | Issues related to custom entities (mobs, NPCs, etc.). |
| `lanterns` | Issues specifically involving the lanterns themselves. |
| `wontfix` | Issues that have been reviewed and decided against for various reasons. |

---

## Reporting Rules
To help us resolve issues quickly, please follow these rules when posting:

1. **Be Respectful:** We are a community. Please use polite language and be patient with maintainers and other contributors.
2. **Check for Duplicates:** Before opening a new issue, use the search bar to see if someone else has already reported it.
3. **Provide Details:** 
   - For **Bugs**: Include your Minecraft version, Forge/Fabric version, and any relevant crash reports or logs.
   - For **Suggestions**: Clearly explain why the feature would be a good addition.
4. **Real Issues Only:** Please avoid "fake" posts, spam, or off-topic comments. The tracker is for technical issues and project-related discussions only.
5. **Use the Right Label:** If you are unsure, don't worry! We will categorize it for you, but trying to match the labels above helps us a lot.

Thank you for being part of the Horrifying Lanterns community!
